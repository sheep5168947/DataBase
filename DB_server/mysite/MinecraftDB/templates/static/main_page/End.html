終界

終末之界（終界，The End）是類似於外空間的黑暗維度，它漂浮在虛空之中。

前往

一個終界傳送門房間。

要進入終末之界，玩家必須在要塞里找到並激活終界傳送門。
終界傳送門房間有3×3的終界傳送門框架（邊角沒有終界傳送門框架），共計有12個終界傳送門框架。有一條階梯通往浮在熔岩池上方的傳送門。在階梯旁邊有一個蠹魚生怪磚。
玩家需要把12個終界之眼分別插入到12個終界傳送門框架中。所有終界傳送門框架都被插入終界之眼後，終界傳送門便會激活並且會發出巨大的轟鳴聲。任何在終界傳送門框架中間3×3空間裡包括基岩和其他終界傳送門框架在內的方塊、實體都會被摧毀並被終界傳送門方塊代替。玩家在進入激活的終界傳送門時會被立刻傳送到終末之界。

環境

終界城。

終末之界由一個大島嶼和遠處的許多小外島構成，它們主要由終界石組成。中央島嶼和外島之間的缺口是一個長約1000格的虛空。 終末之界的天空和虛空都是沒有圖案的靜止黑暗。終末之界沒有日夜交替，與地獄一樣；取而代之的是恆久不變的暗淡光芒。 大多數的物品和方塊在終末之界的功能與在主世界一樣，除了以下例外：

  • 像地獄那樣，在終末之界使用床時會爆炸，其爆炸威力比TNT大25%。並且會生成火。
  • 像地獄那樣，熔岩在終末之界會流得更快些；但它在終末之界流動的距離與主世界一樣是3格。
  • 水在終末之界的特性與在主世界相同。
  • 指南針不能找到出生點，因為玩家不能在終末之界設定新的出生點。
  • 時鐘不能確定太陽和月亮的位置，因為在終末之界中沒有日夜交替，所以時鐘會隨機旋轉。
  • 地獄傳送門不能在終末之界激活。（儘管可以通過指令或外部軟體和編輯器放置）
  • 火會在基岩上面無盡地燃燒，但不會自然傳播到毗鄰的基岩方塊上。

生態域

終界由終末之界，終界小島，終界平地，終界高地，終界荒地所組成

  • 終末之界生成在終末之界中以(0,0)為中心1000格為半徑的圓內。終末之界主島在圓心處生成。終末之界的大多數特性都在這個生態域里，例如終界龍、黑曜石柱、終界水晶、5×5生成平台、返回傳送門和20個終界折躍門。
  • 終界小島作為外島的一部分生成，由較大的島嶼和它周圍較小的圓形島嶼及其之間的空隙組成。
  • 終界平地作為外島的一部分生成，每個島嶼的丘陵到邊角周圍的懸崖之間有較小的高度變化。終界城會在這裡生成，但不會生成歌萊樹。
  • 終界高地作為外島的一部分生成，每個島嶼由丘陵組成。該生態域是唯一一個歌萊樹和終界城都會生成的生態域。
  • 終界荒地作為外島的一部分生成，每個島嶼由延伸的邊角組成，邊角下方是陡峭的懸崖。該生態域不會生成歌萊樹和終界城。
