地獄

Edit

地獄（The Nether）是一個地獄般的維度，充斥著火、熔岩以及危險而強大的生物。

前往

一個地獄傳送門方塊。

要到達地獄，玩家必須在主世界建造一個地獄傳送門（位於終末之界的地獄傳送門並不會被激活）。要創建地獄傳送門，必須建造一個黑曜石框架，大小最小為4×5，最大為23×23。一旦建造好黑曜石框架後，在黑曜石的內側里用打火石或烈焰彈點火（或者使用其他可以使黑曜石著火的方法）就可以激活地獄傳送門，只要黑曜石內側出現火，傳送門就會被激活。地獄傳送門激活後，會發出像終界使者的粒子效果似的紫色粒子。
激活的傳送門允許大多數實體通過（除了凋零怪、終界龍和騎乘著實體的實體或被騎乘著的實體），且會在地獄的相應坐標創建一個新的傳送門（地獄中1格相當於主世界8格），如果相應坐標及其附近已有傳送門，將把實體直接傳送到已有的傳送門，而不是重新創建。如果傳送門創建在空中或液體上方，將會向傳送門兩邊分別突出2個黑曜石來保證實體不會掉下去。以上傳送門生成規則無論是主世界到地獄還是地獄到主世界都有效。黑曜石框架被破壞、傳送門附近爆炸、框架里被桶或發射器放置熔岩或水、地獄幽靈的火球都會導致地獄傳送門關閉。

環境

地獄要塞的內部。

地獄的地形的組成部分大部分是地獄石，它們形成了巨大而光滑的突起和牆壁。熔岩海、熔岩瀑布和熔岩河在地獄會經常出現。藏在地獄石里的熔岩也是很常見的，這讓大型採掘項目變得很危險。在地獄石上永久燃燒的火焰對於粗心的玩家來說也是另一種危險。

螢光石倒掛在洞頂，蘑菇大量地生長在地上。有許多浮在空中的小塊礫石和嵌在地表上的靈魂砂。在地獄石層中存在著小隧道，裡面隨機地散布著地獄石英礦。在靠近熔岩海平面附近的地方，岩漿塊會零星地生成在地獄石中，使得在這裡的通行變得困難，並且盔甲很可能會損壞。

地獄沒有晝夜交替和天氣。僅有的天然光源來自於火、熔岩和螢光石。即使亮度等級為0，地獄仍然中有一層昏暗的光（相當於主世界亮度等級低於8時的樣子）。光照系統其他功能都如它在終末之界和主世界那樣正常運行。

生態域

地獄生態域在生成地獄時使用。諸如地獄幽靈、成群的殭屍豬人、少見的岩漿立方怪和終界使者之類的生物在此生成。地獄石英、螢光石堆、地獄要塞等結構是這裡的專屬結構。雖然水無法在地獄維度中放置，在超平坦世界中選中地獄生態域時，冰和水體湖泊（以及其他的主世界結構）仍然會生成。
